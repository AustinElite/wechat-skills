# wechat-title-hook-generator

## 作用
用于为微信公众号文章生成标题、开头、摘要、封面短句与转发文案。

## 何时使用
当用户出现以下意图时使用：

- 起标题
- 改标题
- 写开头
- 写导语
- 写摘要或封面文案
- 想提高打开率但不想标题党

## 输入优先级
尽量从上下文获取：

1. 文章主题
2. 读者类型
3. 文风偏好
4. 是否偏理性、故事、观点、犀利或温和
5. 是否需要避开夸张表达

## 工作流程
1. 先判断文章属于哪一类：
   - 信息增量型
   - 观点判断型
   - 案例复盘型
   - 故事叙事型
   - 方法教程型
2. 基于文章类型生成不同风格的标题，而不是只换同义词。
3. 至少输出 3 组内容包，每组包含：
   - 标题
   - 开头 2-4 句
   - 摘要
   - 封面短句
4. 明确每组适合的风格：
   - 稳健传播
   - 强冲突打开
   - 深度观点
   - 温和故事

## 输出格式
### 1. 标题方向判断
- 文章类型：
- 最适合的标题策略：

### 2. 内容包
每组包含：
- 标题
- 开头
- 摘要
- 封面短句
- 适用场景

### 3. 推荐版本
- 我最推荐的版本
- 推荐原因
- 如果继续优化，下一步该改什么

## 质量要求
- 标题要有辨识度，但不能骗点开
- 开头第一屏要尽快建立冲突、问题或读者收益
- 摘要不能重复标题，要补充信息
- 同一轮输出要保证风格差异明显

## 禁止事项
- 不要只给标题不给应用场景
- 不要输出低质鸡汤式表达
- 不要为了刺激点击而扭曲原文观点
