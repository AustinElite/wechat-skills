# wechat-style-polisher

## 作用
用于微信公众号文章的改稿、润色、重写、压缩、扩充与风格统一。

## 何时使用
当用户出现以下意图时使用：

- 帮我润色
- 帮我改稿
- 写得太干了，顺一下
- 改成更像公众号
- 太啰嗦了，帮我精简
- 保留原意但让它更能读下去

## 输入优先级
优先识别：

1. 原稿
2. 用户想保留的内容
3. 用户不想要的风格
4. 文章当前问题
5. 是否要保持作者本人口吻

## 工作流程
1. 先诊断原稿问题，再动笔改。
2. 至少从以下几个维度判断：
   - 开头是否抓人
   - 结构是否顺
   - 段落是否过长
   - 观点是否抽象
   - 是否有重复表达
   - 结尾是否无力
3. 改稿时优先保留作者原意和个性表达，只修表达效率与阅读体验。
4. 如果原稿问题很重，先给“重写版”；如果原稿基础不错，给“润色版”。

## 默认输出格式
### 1. 诊断
- 原稿最影响阅读的 3 个问题
- 应该保留的优点

### 2. 改写结果
直接给出完整可用版本。

### 3. 修改说明
- 我重点改了什么
- 为什么这样改
- 如果还要更偏某种风格，可以往哪边推

## 质量要求
- 修改应有针对性，不能只是替换同义词
- 优先解决结构、节奏、信息密度问题
- 对有辨识度的句子谨慎改动，避免把作者个性磨平
- 当用户指定某种风格时，要显式贴近，而不是停留在泛泛“更通顺”

## 禁止事项
- 不要把所有内容都改成同一种模板腔
- 不要无依据地新增事实
- 不要为了“高级感”牺牲清晰度
